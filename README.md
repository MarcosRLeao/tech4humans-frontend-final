# Tech4Humans - Frontend

Este projeto foi desenvolvido com [Create React App](https://github.com/facebook/create-react-app).

## Comandos

No diretório do projeto, execute:

### `npm install`

Para baixar as dependencias

```
npm install
```

### `npm start`

Para iniciar o sistema

```
npm start
```
